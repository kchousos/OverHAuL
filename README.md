This project uses *LLMs* to automatically generate *fuzzing harnesses* for your
C/C++ project.

## Installation

### Dependencies

- [Poetry](https://python-poetry.org/)

    ```
    pipx install poetry
    ```

### Installation Steps

1. Clone the repository:

    ```
    git clone https://github.com/kchousos/<repo-name>.git
    ```

2. Install dependencies:

    ```
    poetry install
    ```
